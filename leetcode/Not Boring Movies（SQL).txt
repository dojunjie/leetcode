X市开了一家新电影院，很多人都想去这家电影院。电影院还会发一张海报，说明电影的评级和描述。
请编写一个SQL查询来输出具有奇数编号ID和非“无聊”描述的电影。按评级排序结果。

例如，表格cinema：

+---------+-----------+--------------+-----------+
| id | 电影| 描述| 评级|
+---------+-----------+--------------+-----------+
| 1 | 战争| 伟大的3D | 8.9 |
| 2 | 科学| 小说| 8.5 |
| 3 | 爱尔兰语 无聊| 6.2 |
| 4 | 冰歌| 幻想| 8.6 |
| 5 | 门卡| 有趣| 9.1 |
+---------+-----------+--------------+-----------+
对于上面的示例，输出应为：
+---------+-----------+--------------+-----------+
| id | 电影| 描述| 评级|
+---------+-----------+--------------+-----------+
| 5 | 门卡| 有趣| 9.1 |
| 1 | 战争| 伟大的3D | 8.9 |
+---------+-----------+--------------+-----------+


SELECT * FROM cinema WHERE (id % 2 = 1) AND (description <> 'boring') ORDER BY rating DESC;
